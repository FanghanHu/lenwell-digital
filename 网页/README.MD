# 屏幕维修

屏幕维修的说明界面：
iPhone, iPad, Apple Watch, Samsung Tablet, Samsung Cellphone 各自需要一个页面， 每个页面需要一个300字左右的解说， 解说内容作为meta使用。不同页面之间的解说不可以重复或者过于相似， 以防止被 Google 认为是在刷关键词。
